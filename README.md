# Alshifa Khan — Portfolio

Personal portfolio website for **Alshifa Khan**, a Computer Science graduate (COMSATS University Islamabad, Abbottabad Campus, 2026) transitioning into **Project Management, Project Coordination, and PMO** roles.

**🔗 Live site:** [alshifaportfolio.vercel.app](https://alshifaportfolio.vercel.app/)

---

## About

Built to highlight project planning, coordination, and delivery experience — not just code. The site frames real work (a Final Year Project and several freelance builds) through a project-management lens: milestones, ownership, and outcomes.

**Sections:**
- **Hero** — intro, current status, quick links (Resume, GitHub, LinkedIn)
- **About** — background and a "profile brief" summary of focus areas
- **Timeline** — an interactive milestone timeline of education, the FYP, and freelance work
- **Projects** — featured Final Year Project (*Job Sensei – AI Career Coach*) plus other builds
- **Skills** — PM toolkit and technical skills, grouped by category
- **Contact** — email, phone, WhatsApp, and LinkedIn

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- Fonts via `next/font`: Fraunces (display), Inter (body), JetBrains Mono (labels/data)
- Deployed on [Vercel](https://vercel.com/)

## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/alshifakhan1411/alshifaportfolio.git
cd alshifaportfolio
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Project Structure

```
app/
  layout.tsx     → root layout, fonts, metadata
  page.tsx       → main portfolio page (all sections)
  globals.css    → design tokens, theme, animations
public/
  profilepicture.png
  ALSHIFA_RESUME.pdf
```

## Deployment

This project auto-deploys to [Vercel](https://vercel.com/) on every push to `main`.
